# Mambo Tools Development Web App
### Installation
1. Clone Repository
2. Install Dependencies _(requires Node.js version 16+)_
```bash
npm install
```
3. Run the Node.js Web Server _(Express.js)_
```bash
npm start
```
4. To run a new Build process _(./build)_
```bash
npm run build
```
